# 🔍 Context Lens for X (Twitter)

**Context Lens** is a production-grade Chrome extension that brings AI-powered insights directly to your X.com (Twitter) timeline. It analyzes posts in real-time to detect AI-generated content, sentiment, toxicity, and provides concise summaries—all within a native-looking interface.

---

## 🚀 Features

- **AI Probability Detection**: Identify posts likely generated by LLMs.
- **Sentiment & Tone Analysis**: Understand the emotional context of a post.
- **Toxicity Shield**: Highlight inflammatory or harmful content.
- **Smart Summarization**: Get the gist of long posts instantly.
- **Dual-Mode Engine**: Choose between high-accuracy API analysis (Claude/GPT) or fast, private Local analysis.
- **Native Integration**: Matches X's design system (dark & light modes).
- **Infinite Scroll Support**: Automatically detects and analyzes new posts as you scroll.
- **Privacy-First**: Optional local mode ensures no data leaves your browser.

---

## 🛠️ Installation

### Developer Installation (Load Unpacked)

1. **Clone/Download** this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **"Developer mode"** (top right corner).
4. Click **"Load unpacked"**.
5. Select the root folder of this project (`context-lens-extension`).
6. Navigate to [x.com](https://x.com) to see the lens in action!

---

## ⚙️ Configuration

Open the extension popup by clicking the icon in your toolbar to:
- Toggle specific analysis features.
- Switch between **API Mode** and **Local Mode**.
- Provide your **Anthropic API Key** for advanced insights.
- Clear the local analysis cache.

## 🏗️ Technical Architecture

### Component Breakdown
- **Content Scripts**: `MutationObserver` monitors X's dynamic React-based DOM.
- **Background Worker**: Handles asynchronous API requests and coordinates caching.
- **Storage**: Multi-layer cache using `chrome.storage.local` with TTL management.
- **Analysis Engine**: Modular system for plug-and-play analysis components.

### Performance Optimization
- **Debounced DOM Scanning**: Prevents CPU spikes during rapid scrolling.
- **WeakSet Tracking**: Eliminates duplicate processing of previously seen tweets.
- **Skeleton Loaders**: Provides immediate visual feedback while analysis is pending.
- **CSS Containment**: Minimizes layout shifts on x.com.

---

## 🛡️ Security & Privacy
- **No `eval()`**: Fully compliant with Manifest V3 security policies.
- **Sanitized Injection**: Uses `textContent` and safe DOM manipulation to prevent XSS.
- **Encrypted Storage**: API keys are stored securely within Chrome's local storage area.
- **Zero Tracking**: No personal data (PII) is collected or sent to external trackers.
